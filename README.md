# Asteroids

##############Таблица рекордов############
Сохраняет заработанные очки, который видно сверху по середине во время игры.
Посмотереть таблицу рекордов можно во время игры нажав “Tab".
Выйти можно нажав “Space” или “Enter”
Рекорды сохраняются по прохождеиню уровней и выхода из игры.
############### УПРАВЛЕНИЕ ###############
Стрелки влево, вправо - поворот коробля
Стрелка вверх - движение
Пробел - выстрел
##################Уровни##################
Есть 3 уровня.
###############Игровые объекты############
Красная аптечка лечит.
Синяя аптечка дает 3 выстрела за раз на 3 выстрела.
Черные дыры притягивают.
Астероиды при столкновении наносят урон.
Пришельцы стреляют.
###################Читы###################
Кнопка "a" или "ф" дает ХП(неограниченно).
Кнопка "s" или "ы" дает синии апетчки.
###################Пасхалка###################
Нажмите кнопку "n" или "т" и на следующем уровне поменяется скин у астероидов.
Мем с неграми и гробом.
(Музыка тоже из мема)
